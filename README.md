# CNN-Based Potato Disease Detection

## Project Overview
This project uses a Convolutional Neural Network (CNN) model to identify potato plant diseases from images. The prediction results are displayed through the Blynk IoT platform for remote monitoring.

## Features
- Potato disease detection
- Image classification using CNN
- Real-time prediction
- Blynk dashboard integration
- Remote monitoring

## Technologies Used
- Python
- TensorFlow/Keras
- Blynk
- HTML
- Machine Learning
- Deep Learning

## Project Files
- main_python_script.py
- convert_model.py
- index.html (it was also hosted here on streamlit {web interface} )
- my_potatoes_trained_model
- screenshot of my process

## How It Works
1. The user uploads or captures an image of a potato leaf.
2. The CNN model analyzes the image.
3. The disease prediction is generated.
4. The result is sent to Blynk and displayed on the dashboard {app interface}.
